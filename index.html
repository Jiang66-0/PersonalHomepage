March 3, 2026 - Individual Project

Q-Learning Maze Game

Problem: Train a reinforcement learning agent to autonomously find the optimal path in a fixed 10x10 maze with walls, without prior knowledge of the layout.

Data: A custom 10x10 grid maze environment with 100 discrete states, 4 possible actions, and a designed reward system for agent learning.

Approach: Implemented the classic Q-Learning algorithm with an epsilon-greedy exploration strategy, using NumPy for Q-table management and Pygame for maze visualization and interactive UI.

Outcome / Impact: The agent successfully learned the optimal path after 3,000 training episodes and could stably navigate from the start to the goal without collisions or infinite loops.

My Contribution: Independently designed the maze environment, implemented the full Q-Learning algorithm, built the reward function, optimized the exploration rate and training parameters, and developed the visualization interface.

April 10, 2026 - Group Project

Deep Learning for Fake News Detection: A Comparative Study of BiLSTM-Attention and TextCNN

Problem: Address the issue of rampant fake news on social media by developing an automated fake news detection system based on deep learning for binary classification of news text.

Data: A real-world dataset containing 4,749 labeled news articles with text content and classification labels, with a nearly balanced data distribution between real and fake news.

Approach: Built a complete NLP pipeline including data cleaning, text normalization, tokenization, vocabulary construction, and sequence encoding, then developed and compared BiLSTM-Attention and TextCNN models with multiple hyperparameter settings.

Outcome / Impact: The BiLSTM-Attention model achieved 74.61% test accuracy and 0.8203 ROC-AUC, while TextCNN performed slightly better with 75.04% accuracy. The project produced a reproducible deep learning text classification pipeline and a standardized solution for lightweight fake news detection.

My Contribution: Responsible for data preprocessing and exploratory data analysis, including cleaning, text normalization, denoising, URL and symbol removal, tokenization, vocabulary construction, sequence encoding, dataset splitting, text length statistics, and distribution visualization.
